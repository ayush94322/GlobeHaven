# GlobeHaven 🌍

GlobeHaven is a full-stack travel web application that allows users to discover unique places to stay around the world, create and manage property listings, upload images, and share reviews — all in a smooth, secure, and responsive experience.

---

## Features

### User Authentication & Authorization
- Secure user registration (Sign up)
- User login with session-based authentication
- User logout functionality
- Protected routes with authorization middleware
- Flash messages for better UX

### Property Listings Management
- **Create** new property listings with details and images
- **Edit** existing listings
- **Delete** listings
- **View** all listings with detailed information
- Upload and manage multiple listing images via **Cloudinary**

### Reviews & Ratings System
- Add ratings and reviews for listings
- View all reviews for a specific property
- User-specific review management

### Maps Integration
- Interactive maps using **Mapbox API**
- Display property locations on the map
- Visual property exploration

### Payment Gateway Integration
-Secure and reliable payment system integrated
-Accepts domestic and international payment both
-All type of payment upi, cards, qr-based .etc accepted

### Responsive Design
- Mobile-friendly UI built with **Bootstrap**
- Seamless experience across all devices

---

## Key Highlights

- Built a **full-stack web application** using **MVC architecture**
- Implemented **authentication & authorization** using **Passport.js**
- Integrated **Cloudinary** for secure and efficient image uploads
- Designed **RESTful APIs** with **Express.js**
- Integrated interactive maps using **Mapbox API**
- Used **MongoDB + Mongoose** for robust data modeling
- Applied **server-side validation** using **Joi**
- Responsive and dynamic UI built with **Bootstrap & EJS**
